# Desafio8Rainhas
 Desafio das 8 rainhas

# --------------------------------------------------------------------- #
#			DESAFIO DAS OITO RAINHAS			#			
# --------------------------------------------------------------------- #

Joguinho simples feito com JavaScript Puro, as únicas importações efetuadas
está relacionado aos ícones ( https://fontawesome.com/ ).

O objetivo do jogo é simples, adicione oito rainhas no tabuleiro de xadrez
sem que uma rainha consiga matar outra, para isso clique sobre a célula do
tabuleiro que deseja adicionar. Caso uma rainha tenha acesso a outra rainha
em um de seus movimentos básicos, ambas ficarão destacadas e o sistema não
permitirá adicionar outra até que uma delas deja removida.
